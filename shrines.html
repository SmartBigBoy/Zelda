☁️ 空岛神庙

空岛是游戏的序章区域，包含教学性质的神庙。以下为核心神庙攻略：

类型：教学 难度：★

  1. 将石块融合到树枝上，获得强化武器
  2. 用强化武器击碎石墙
  3. 将火箭融合到盾牌上，按ZL+A发射升空
  4. 滑翔到出口，完成神庙
余料建造是本作核心能力，务必掌握各种素材的融合效果。

类型：教学 难度：★

  1. 站在木质平台下方，使用通天术穿越到上层
  2. 连续穿越多层平台到达顶层
  3. 获取宝箱后前往出口

类型：教学 难度：★

  1. 观察滚动的石球，使用倒转乾坤使其倒退
  2. 利用移动平台的倒转运动到达高处
  3. 让滚石倒退创造安全通道

类型：教学 难度：★

  1. 用究极手抓取木板，搭建跨越断崖的桥梁
  2. 将物体组合在一起创建更大的结构
  3. 利用组合物体到达对岸

🏔️ 初始台地及周边

类型：解谜 难度：★★

  1. 用究极手将铁板搭在水面上形成桥梁
  2. 将多个铁板组合在一起跨越更宽的水域
  3. 利用铁球滚动机关开启大门

类型：射击 难度：★★

  1. 用弓箭射击移动靶标
  2. 注意靶标的移动节奏，提前瞄准
  3. 全部击中后开启出口
使用余料建造将眼球融合到箭矢上，可以实现追踪射击，大幅降低难度。

🌾 海拉鲁平原

海拉鲁平原区域包含大量神庙，以下为部分重点神庙：

类型：战斗 难度：★★

  1. 使用侧跳闪避敌人攻击，触发林克时间
  2. 利用盾反反弹敌人投掷物
  3. 击败所有守护者后完成
战斗神庙内武器固定，不需要自带武器。利用场地内的柱子和障碍物做掩体。

类型：解谜 难度：★★

  1. 观察升降平台的运动轨迹
  2. 站在下行的平台上，使用倒转乾坤使其上升
  3. 到达顶层获取宝箱和出口

类型：解谜 难度：★★

  1. 使用通天术穿越天花板到达上层
  2. 利用水面的倒影判断穿越位置
  3. 连续穿越多层后到达出口

🌬️ 利特地区（风）

类型：解谜 难度：★★

  1. 利用风场上升气流到达高处平台
  2. 在空中滑翔穿过环形目标
  3. 用究极手调整风向标志改变气流方向
  4. 到达最高处出口

类型：解谜 难度：★★★

  1. 用究极手旋转风车装置，产生上升气流
  2. 乘气流到达上层，调整第二个风车方向
  3. 组合多个风车的气流，到达最终出口

🔥 鼓隆地区（火）

类型：解谜 难度：★★★

  1. 用究极手将矿车放到轨道上
  2. 给矿车安装火箭作为动力源
  3. 乘坐矿车沿轨道前进，途中用弓箭击落障碍
  4. 到达终点完成神庙
矿车行驶速度较快，注意不要掉落。可以用倒转乾坤让矿车倒回。

类型：解谜 难度：★★★

  1. 用究极手调整炮台角度
  2. 装入炮弹发射，炸开堵路的岩石
  3. 调整炮台方向摧毁远处的目标
  4. 获取宝箱后前往出口

💧 卓拉地区（水）

类型：解谜 难度：★★★

  1. 用究极手调整水渠方向，引导水流驱动水车
  2. 水车转动后打开闸门
  3. 利用水流冲刷移动平台到达对岸
  4. 调整最终水流方向完成解谜

⚡ 格鲁德地区（雷）

类型：解谜 难度：★★★★

  1. 利用金属物体作为导体，将电力从电源传导到目标
  2. 用究极手移动金属板搭桥连接电路
  3. 注意电流方向和断路点
  4. 完成电路连接后开启机关
导电解谜中，金属物体和左纳乌导电装置都可以传导电流。注意非金属物体不导电。

类型：解谜 难度：★★★★

  1. 用究极手调整镜子的角度，将光线反射到目标位置
  2. 利用多个镜子连续反射光线到达远处
  3. 注意光线必须精确照射到接收器上
  4. 完成后开启出口

🗻 拉聂尔地区

类型：解谜 难度：★★★

  1. 用究极手将车轮、操纵杆和平台组合成载具
  2. 驾驶载具穿越障碍地形
  3. 在断崖处加装风扇提供飞行动力
  4. 到达终点完成

🌴 费罗尼地区

类型：解谜 难度：★★★

  1. 利用重力球从高处滚落触发机关
  2. 用究极手调整坡道角度控制球的方向
  3. 将球引导到指定位置开启大门

🍂 阿卡莱地区

类型：解谜 难度：★★★★

  1. 观察齿轮的转动方向和速度
  2. 使用倒转乾坤逆转齿轮运动
  3. 利用逆转的齿轮作为电梯到达上层
  4. 组合多个逆转机关完成最终解谜

🌑 地底神庙

地底的神庙与地表神庙位置一一对应（镜像），但内部解谜完全不同。地底神庙的特点是黑暗环境，需要发光花或暗视药辅助。

进入地底前，建议准备足够的发光花和暗视药。地底敌人较强，推荐至少10颗心之容器以上再深入探索。

地底神庙通用技巧

  • 发光树：每棵发光树对应地表一座神庙，以树为路标
  • 暗视药：暗夜萤火虫+怪物素材，临时增加视野范围
  • 发光花：蓝色花串引导方向，跟随花串找到神庙
  • 明亮蘑菇：地面散布的发光蘑菇标记了安全路径
  • 左纳乌装置：地底有大量可用装置，善加利用
