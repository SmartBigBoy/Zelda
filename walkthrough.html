序章：空岛觉醒

🟢 觉醒室

游戏开场，林克在空岛的觉醒室中醒来。跟随引导完成基础操作学习：

  1. 起身走出觉醒室，沿路前进，学习移动、跳跃和攀爬
  2. 获得究极手能力 — 拾取、移动、旋转、组合物体
  3. 利用究极手将木板搭桥，跨越断裂的道路
  4. 继续前进，学习战斗基础（拾取树枝攻击木桩敌人）
究极手是本作最核心的能力，可用于搭建桥梁、制作载具、组合武器等，务必熟练掌握。

🟢 古塔尼罗神庙 - 余料建造

进入神庙学习余料建造能力：

  1. 将石块融合到树枝上，大幅提升攻击力
  2. 用融合后的武器击碎堵路的石墙
  3. 将火箭融合到盾牌上，按住ZL+A发射升空
  4. 在空中打开滑翔伞到达出口
余料建造可以融合几乎所有素材到武器/盾牌/箭矢上。不同素材提供不同效果，例如：怪物角增加攻击力，冰/火素材赋予元素属性，火箭盾可紧急升空。

🟢 古乔尼神庙 - 通天术

学习通天术能力，可穿越天花板：

  1. 站在木质平台下方，使用通天术穿到上层
  2. 利用通天术穿过多层平台到达顶部
  3. 在水下方的平台使用通天术穿越到水面上的房间
  4. 获取宝箱后前往出口
通天术只能在有实体天花板时使用，且上方需要有足够空间。如果上方被堵住或没有平台，则无法使用。

🟢 纳乔亚哈神庙 - 倒转乾坤

学习倒转乾坤能力，让物体沿运动轨迹倒退：

  1. 观察滚动的石球，使用倒转乾坤使其倒退回原位，趁机通过
  2. 让上升的平台上移，利用其返回时站上去到达高处
  3. 让滚石倒退，制造安全通道
倒转乾坤在战斗中也非常实用：可以让投掷的炸弹桶飞回敌人手中，让滚落的巨石退回高处再释放砸敌。

🟢 离开空岛

获得四大能力后，前往空岛边缘的跳台，跳向下方发光的池塘，开启滑翔，正式进入海拉鲁大陆。

  1. 前往时间神殿废墟，触发剧情
  2. 从空岛最外侧的跳台跳下
  3. 在空中打开滑翔，朝地表的发光池塘飞去
  4. 落入水池后，抵达海拉鲁大陆
空岛序章约需30-60分钟完成，期间无法返回地表。建议在此期间充分练习四大能力的使用。

第一章：前往地表

📍 监视堡垒

降落地表后，首要目标是前往监视堡垒：

  1. 从降落地出发，跟随标记前进
  2. 途中遇到霍斯塔，他会指引你去监视堡垒
  3. 到达监视堡垒，与普尔亚对话
  4. 完成前置任务，解锁塔台功能（地图解锁）
  5. 激活第一个塔台 — 开启海拉鲁大陆探索
监视堡垒是前期的核心据点，有商店、旅馆、料理锅等重要设施。建议以此为基地，逐步向外探索。

📍 解锁关键功能

在监视堡垒及其周边，建议优先解锁以下功能：

  • 相机功能：与罗贝利对话，修复普尔亚平板的相机模块
  • 滑翔伞升级：完成风之神殿后获得
  • 驿站网络：登记马匹，解锁快速旅行点
  • 蓝图功能：在监视堡垒下方地底与乔修对话获取

第二章：地区异变

四大地区各有一座神殿需要解放，推荐按以下顺序挑战：风→火→水→雷（由易到难）。

🌬️ 风之神殿（利特村）推荐首选

前置任务：前往利特村，与丘栗对话开启任务。

前往利特村

  1. 从监视堡垒出发，向西北方向前往塔邦挞大桥
  2. 经过塔邦挞雪原，注意保暖（防寒服或辣味料理）
  3. 到达利特村，与丘栗会合
  4. 跟随丘栗乘飞行器前往风之神殿入口

风之神殿解谜

  1. 第一把锁：使用究极手转动风车机关，打开第一道门
  2. 第二把锁：利用上升气流到达高处平台，用通天术穿过天花板
  3. 第三把锁：用究极手移动挡风的障碍物，让风力驱动风车
  4. 第四把锁：在旋转平台间跳跃，注意利用滑翔和上升气流
  5. 第五把锁：击败守护者敌人，解锁最后一把锁

Boss：急冻盖拉

风之神殿的守护Boss，弱点为火属性攻击。

  • 阶段一：Boss在冰柱上释放冰弹，用火箭射击冰柱使其坠落
  • 阶段二：Boss飞到空中释放冰柱阵列，用究极手投掷炸弹或射箭攻击
  • 阶段三：Boss坠落后近身攻击，用火焰武器连续输出
装备火焰武器或提前准备火焰果箭可大幅提升战斗效率。丘栗的横向暴风能力可帮助接近空中的Boss。

奖励

  • 丘栗的誓约（横向暴风：创造上升气流和横向暴风）
  • 1颗心之容器
  • 滑翔伞布料升级

🔥 火之神殿（鼓隆城）需防火装备

前置任务：前往鼓隆城，与阿沅对话。

前往鼓隆城

  1. 从监视堡垒向东北出发，前往死亡之山
  2. 在山脚下购买防火药或穿戴耐火铠
  3. 沿路上山，经过死亡之山登山道
  4. 到达鼓隆城，发现城内被莫力布林占领
  5. 与阿沅合作，用炮弹清除城内敌人

火之神殿解谜

  1. 第一把锁：用究极手将矿车推上轨道，乘坐矿车前往第一区域
  2. 第二把锁：用炮台炸开堵路的岩石，注意炮弹的角度
  3. 第三把锁：在岩浆区域利用冷却的岩石跳跃前进
  4. 第四把锁：启动矿车轨道系统，到达深层区域
  5. 第五把锁：击败火焰魔像，解锁最后机关

Boss：熔岩巨像

火之神殿的守护Boss，弱点为冰属性攻击。

  • 用冰箭或冰霜武器击打Boss暴露的核心
  • Boss倒地后近身输出
  • 注意躲避Boss的岩浆喷射和砸地攻击
阿沅的滚石攻击可对Boss造成大量伤害。提前准备冰霜武器或冰冻花箭。

奖励

  • 阿沅的誓约（滚石：向前滚动的大石球攻击）
  • 1颗心之容器

💧 水之神殿（卓拉领地）需防水装备

前置任务：前往卓拉领地，与希多对话。

前往卓拉领地

  1. 从监视堡垒向东出发，前往拉聂尔地区
  2. 在河边找到卓拉族的信使，跟随指引
  3. 注意：前往卓拉领地需要经过大量水域，准备防水装备
  4. 到达卓拉领地，与希多王子对话
  5. 完成前置任务：解决水源污染问题

水之神殿解谜

  1. 第一把锁：操控水位开关，露出隐藏通道
  2. 第二把锁：用究极手调整水流方向，驱动水车
  3. 第三把锁：利用希多的水盾能力穿越激流区域
  4. 第四把锁：在水下迷宫中找到出口，注意空气槽
  5. 第五把锁：击败水生魔物，解除最后封印

Boss：污泥巨蛇

水之神殿的守护Boss。

  • Boss在污泥中移动，需要先清除污泥（用水或希多的水盾）
  • 清除污泥后Boss核心暴露，趁机攻击
  • 注意Boss的冲撞攻击和污泥喷射

奖励

  • 希多的誓约（水盾：创造水之护盾，可防御和冲浪）
  • 1颗心之容器

⚡ 雷之神殿（格鲁德小镇）难度最高

前置任务：前往格鲁德沙漠，与璐珐对话。

前往格鲁德小镇

  1. 从监视堡垒向西南出发，前往格鲁德沙漠
  2. 需要穿女装才能进入格鲁德小镇（在卡拉卡拉集市购买）
  3. 到达格鲁德小镇后，与璐珐对话
  4. 完成前置任务：调查沙暴的来源
格鲁德沙漠白天酷热、夜晚极寒，请准备耐热和防寒装备/料理。沙暴中会迷失方向，建议携带定位标记。

雷之神殿解谜

  1. 第一把锁：使用通天术穿越地面到达地下区域
  2. 第二把锁：利用导电机制，将电力传导到指定位置
  3. 第三把锁：在黑暗迷宫中找到出路，注意闪电陷阱
  4. 第四把锁：操控光线反射机关，照亮目标区域
  5. 第五把锁：击败雷电魔像，解除最终封印

Boss：女王吉波

雷之神殿的守护Boss，难度较高。

  • Boss会召唤雷电攻击，注意观察落雷标记并远离
  • 利用璐珐的雷击能力反制Boss
  • Boss分身阶段：找到本体（唯一会投射影子的那个）
  • Boss血量低时会进入狂暴，攻击频率加快，保持防御
装备雷鸣头盔可免疫雷电伤害，大幅降低战斗难度。如暂时没有，可使用绝缘药剂替代。

奖励

  • 璐珐的誓约（雷击：召唤落雷攻击）
  • 1颗心之容器

第三章：寻找大师之剑

🐉 龙之泪收集

收集12个龙之泪是获取大师之剑的前置条件。龙之泪分布在海拉鲁大陆各处的地画位置。

龙之泪位置（推荐顺序）

编号 位置 区域 寻找要点
1监视堡垒附近海拉鲁平原序章自动指引
2塔邦挞雪原西北注意防寒
3奥尔汀峡谷东北注意防火
4拉聂尔湿地东部水域较多
5费罗尼草原东南较为简单
6格鲁德沙漠西南耐热装备
7海布拉山区北部高海拔注意防寒
8阿卡莱地区东北角偏远，需长途跋涉
9中央海拉鲁中心靠近城堡区域
10湖畔地区南部环境友好
11丘陵地区中西部相对容易
12关键龙之泪解锁后出现收集前11个后自动出现
每触摸一处地画，会触发一段关于塞尔达公主的记忆片段，逐步揭开本作的剧情真相。

⚔️ 获取大师之剑

收集12个龙之泪后，白龙会在海拉鲁上空飞行。飞到白龙头上即可拔取大师之剑。

拔剑方法

  1. 完成12个龙之泪的收集，解锁白龙位置
  2. 确保拥有2圈以上精力槽（或13颗心之容器以上）
  3. 前往白龙经过的空域（推荐从高空塔台跳下）
  4. 滑翔到白龙背上，走向龙头位置
  5. 按A键拔剑 — 需要持续按住，消耗精力
  6. 拔出大师之剑！
如果精力不足，拔剑会失败并掉下龙背。建议提前准备精力恢复药或确保精力槽足够长。
大师之剑基础攻击30，在盖侬污染区域（城堡、地底深处）攻击力翻倍至60。耐久耗尽后10分钟自动恢复，是本作最可靠的武器之一。

第四章：海拉鲁城堡危机

🏰 城堡探索

完成四大神殿和大师之剑获取后，前往海拉鲁城堡调查异变。

  1. 前往海拉鲁城堡入口，触发剧情
  2. 深入城堡地下，发现通往深渊的入口
  3. 在地下通道中击败幻影盖侬
  4. 发现盖侬多夫的真正据点在极深的地底

幻影盖侬战斗

  • 幻影盖侬会使用多种武器攻击，注意闪避和盾反
  • 大师之剑对其有额外伤害
  • 击败后获得重要道具，解锁通往最终区域的路径

终章：讨伐盖侬多夫

👁️ 深入深渊

最终决战位于海拉鲁地底最深处。建议做好充分准备再出发：

推荐准备

  • 心之容器：至少15颗以上
  • 武器：大师之剑 + 多把高攻击力武器
  • 防具：蛮族套装（攻击力提升）或勇者套装
  • 料理：大量回血料理（生命榴莲×5）、攻防Buff料理
  • 同伴：四大贤者的誓约全部装备
  • 精灵：至少携带2-3个精灵作为保险

👹 最终Boss：盖侬多夫

最终战斗分为多个阶段：

阶段一：人形盖侬多夫

  • 盖侬多夫使用长枪攻击，注意侧闪和盾反
  • 利用四大贤者的辅助能力创造输出机会
  • 当Boss蓄力攻击时，闪避触发林克时间
  • 大师之剑造成额外伤害

阶段二：魔王盖侬

  • 盖侬多夫化身为巨大的魔王形态
  • 利用各贤者能力对应的弱点攻击
  • 丘栗的暴风击碎Boss的护盾
  • 阿沅的滚石攻击Boss核心
  • 希多的水盾防御Boss的范围攻击
  • 璐珐的雷击对Boss造成大量伤害

阶段三：龙之决战

  • 白龙与盖侬多夫的黑龙对决
  • 骑在白龙背上，接近黑龙并攻击其背上的眼睛
  • 用弓箭射击所有弱点
  • 最后按提示按键完成终结一击
阶段三不需要消耗自己的武器，使用白龙提供的光箭即可。享受这段史诗般的最终战斗吧！

🎉 通关后

击败盖侬多夫后，观看结局动画。通关后存档会标记星号，你可以继续在海拉鲁大陆自由探索：

  • 完成未完成的神庙（152座全收集）
  • 收集所有克洛格种子（1000个）
  • 完成所有支线任务
  • 探索地底全区域
  • 收集全防具套装
  • 挑战高难度Boss（如白银莱奈尔、吉波女王等）
